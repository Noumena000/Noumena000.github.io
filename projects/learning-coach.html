Learning Coach

Learning Coach is an AI-supported educational project currently under private development. It explores how technology can help learners practice deliberately, reflect on their progress, and build understanding without shortcutting the work of learning.

Status

Active Development
Interface previews and demonstrations will be released as development progresses.

Purpose

To support guided practice and reflection while keeping the learner actively engaged in the process of understanding.

Preview Coming Soon This project is currently under active development. Selected interface previews and demonstrations will be published as the project approaches public release.

Overview

AI support that keeps the learner engaged.

Learning Coach is designed around the idea that useful educational technology should strengthen a learner's own reasoning rather than replace it. The project emphasizes careful pacing, learner agency, and thoughtful support for comprehension.

The project connects instructional design, AI literacy, and human-centered software with a focus on learning as an active process.

Focus areas

Design priorities

  • Guided practice
  • Reflective learning
  • Clear goals and pacing
  • Learner agency
